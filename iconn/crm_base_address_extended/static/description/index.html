CRM Base Address Extended

Provides on-demand import of cities and filters city selection by State to keep address data consistent.

Features

  • City dropdown is filtered by selected State.
  • City dropdown is hidden until State is selected.
  • Wizard to import cities per Country/State on demand.

Data Source

Countries, states, and cities data from countries-states-cities-database (ODbL-1.0). https://github.com/dr5hn/countries-states-cities-database
